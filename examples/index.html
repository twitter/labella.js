Labella.js

Labels should be beautiful.

Tweet  

Long long ago, in a computer not that far away, there was a timeline, with many points on it. A young researcher tried to annonate all those points and struggled to place labels without overlaps. Placing them manually is the last thing they wanted to do.
They spent a lot of time trying different methods without success, until they tapped into the force. Since then, whenever someone asks them how to place the labels, they simply reply, "May the force label you." However, not everybody understood their hint, so they made this library.

Want to try it? Play with the interactive demo below to see how Labella handles randomly generated labels with different parameters.

Want to use it? See a few simple examples to help you start, visit the github repo or read the documentation.
